< Project >

E-commerce website store

Built with HTML, CSS, Javascript and PHP

User authentication

Basic user authentication function to register user account or sign in to existing account. Additionally, AJAX solution were implemented to handle username availability checking before submitting the registration form.

Admin CRUD

Customized product page for adminstrators to handle CRUD operations without implementing any database language statements.

Product search bar

Classic search bar allows user to search for products through the website. Furthur enhance functionality by implementing jQuery .autocomplete function to filter and narrow down searching area.

Customers cart

One of the essential component of e-commerce store function as gather and manage products user wish to purchase. Allow user to track and review their cart before proceeding to checkout.

Behind the website, the data processing and handling processes are driven by SQL statements and facilitated through PHPMyAdmin. The CRUD operations that will be perform by Admins will be done through prepared SQL statement in order to update new records to database. The are store in four table that can be illustrated through ERD diagram below:

To ensure data integrity, Primary key and Foreign key are set up to uniquely identify each records and establish relationship between one column to another table's column.

In this website, a few security measures were implemented to defend some common attacks from user actions to website. Each user inputs will be sanitized using PHP function such as htmlspecialchars() and stripslashes(). Apart from that, the SQL statement and queries will be prepared and parameterized to ensure only valid user inputs can be processed.
